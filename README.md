## About Me

Hello, I am aonekoaoi.

I enjoy C, Java, and Python as a hobby. **It's just a hobby**, so don't expect that much programming skill.

But we would like to distribute source code that will help you as much as possible.

**My programming policy is to waive as many rights as possible to make it easier for everyone to use.**

Therefore, we distribute a lot of source code under the [MIT license](https://github.com/aonekoaoi/LicenseRepositories/blob/main/License/MIT/LICENSE.txt).

こんにちは、青猫あおいと申します。

C, Java, Pythonを趣味程度で嗜んでいます。**趣味程度**ですのでプログラミングスキルに期待しないでください。

ですが少しでもあなたのお役に立てるようなソースコードを配布します。

**私のプログラミングポリシーはできるだけ権利を放棄して誰もが使用しやすくすることだ。**

そのため[MITライセンス](https://github.com/aonekoaoi/LicenseRepositories/blob/main/License/MIT/LICENSE_ja.txt)で多くのソースコードを配布します。
